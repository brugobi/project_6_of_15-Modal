# Modal

> This is the project 6 of 15 from the course created by [John Smilga](https://www.johnsmilga.com) and also avaible on [you tube](https://www.youtube.com/watch?v=c5SIG7Ie0dM&list=PLnHJACx3NwAey1IiiYmxFbXxieMYqnBKF). **Modal** is the implementation of the buttons that when clicked appears the modal content and disappears when you click to close it.

## Built With

- HTML
- CSS
- VSCode
- Vanilla JavaScript

## Getting Started

### Prerequisites

- Any Brownser

To get started open the directory where you will save this repo by typing on your terminal:

```
$ cd <directory>
```

- Clone the repo typing:

```
$ git clone git@github.com:brugobi/project_6_of_15-Modal.git

```

### Usage

- open the file *index.html* in any Brownser
